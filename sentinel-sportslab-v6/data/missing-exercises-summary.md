## Notes for Review

1. **Aliases** (24 exercises) are intentional duplicates — they let users search by common names. Consider whether to insert these as separate exercises or create a search-alias system instead.
2. **Machine exercises** are the highest-priority gap — every commercial gym has these machines.
3. **Foam Roller exercises** use `tracking_type: "time"` since they are typically duration-based.
4. **Cardio machines** (treadmill, bike, etc.) use `tracking_type: "distance"` or `"time"`.
5. All exercises use the Supabase schema fields: `body_parts[]`, `categories[]`, `options{}`, `equipment[]`, `tags[]`, `tracking_type`.
